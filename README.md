# Uncanny Clicker

指定した座標を、指定した回数だけ自動でクリックするPython製のクリックツールです。  

## 機能

- 指定したX座標・Y座標を自動クリック
- クリック回数を指定可能
- 1秒あたり16回クリック
- 2画面環境に対応
- マウスを画面の角に移動すると安全停止
- 実行後に平均クリック速度を表示

## 動作環境

- Windows
- Python 3.x
- pyautogui

## インストール方法

まず、Pythonがインストールされていることを確認してください。

```
python --version
```
次に、必要なライブラリをインストールします。
```
pip install pyautogui
```

clicker_tool.py があるフォルダに移動します。
```
cd C:\Users\owner\Desktop\Uncanny-clicker
```

プログラムを実行します。
```
python clicker_tool.py
```

実行すると、以下の項目を入力します。
```
・クリックするX座標を入力してください:
・クリックするY座標を入力してください:
・クリック回数を入力してください:
```
入力後、3秒後にクリックが開始されます。

## 座標の確認方法

クリックしたい場所の座標を確認するには、Uncanny-clickerフォルダに同封されているmouse_position.py を使用します。
```
python mouse_position.py
```

マウスをクリックしたい場所に移動すると、現在の座標が表示されます。
```
X: 500, Y: 300
```

## クリック速度

初期設定では、1秒あたり16回クリックします。

clicker_tool.py 内の以下の値を変更すると、クリック速度を変更できます。
```
cps = 16
```

例：1秒あたり20回クリックしたい場合
```
cps = 20
```

## 安全停止について

PyAutoGUIには安全停止機能があります。
マウスを画面の角に移動すると、自動クリックが停止します。

以下のようなエラーが出た場合は、安全停止が働いた状態です。

```
pyautogui.FailSafeException
```

クリック位置が画面の端に近すぎる場合にも発生することがあります。
その場合は、少し内側の座標を指定してください。

## 注意事項
- 使用する前に、クリック先の画面を前面に表示してください。
- クリック速度を上げすぎると、クリック先のアプリが反応しきれない場合があります。
- ゲームやWebサービスなどで利用規約に反する使い方はしないでください。
- 暴走した場合は、マウスを画面の角に移動して停止してください。
